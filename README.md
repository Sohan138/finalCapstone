Installation:
To install the Final Capstone project locally, follow these steps:

Clone the repository to your local machine:
git clone https://github.com/your_username/final-capstone.git

Navigate to the project directory:
cd final-capstone

Install dependencies:
npm install

Usage:
After installing the project, you can run it using the following commands:
npm start
This will start the project and you can access it in your web browser at http://localhost:3000.
